## パスワードマネージャー

パスワードマネージャーをシェルスクリプトを活用することで利用できます。
以下の手順に従って利用してください。

1. ”password_manager.sh”ファイルをダウンロードしてください。

2. 上記のファイルを任意のフォルダに格納してください。

3. ターミナルを起動し、cdコマンドで”password_manager.sh”が格納されているディレクトリに移動してください。

4. ターミナルで以下のコマンドを実行して、スクリプトに実行権限を与えます。
```
chmod +x password_manager.sh
```

5. ターミナルで以下のコマンドを入力してスクリプトを実行してください。
```
./password_manager.sh
```

6. 「サービス名」、「ユーザー名」、「パスワード」がそれぞれ聞かれるので入力してください。入力された情報は "passwords.txt" ファイルに追記されます。
